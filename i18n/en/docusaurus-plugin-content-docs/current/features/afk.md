## âš™ï¸ How It Works

### Manual Activation

Type `/afk` to manually enter AFK mode:
- A message will appear in chat about your status
- Your name may display with an **AFK** tag

### Automatic Activation

On some servers, AFK activates automatically if you don't move for a certain time.

### Exiting AFK

AFK mode automatically deactivates when you:
- Move
- Write in chat
- Interact with the world
- Type `/afk` again

When exiting AFK, you'll see a message showing how long you were away.

:::warning Warning
On some servers, players may be kicked for staying AFK too long!
:::

**Why this exists:**
- Freeing slots for active players
- Reducing server load
- Fair resource distribution

**How to avoid being kicked:**
- Move periodically
- Don't leave the game unattended for long periods

:::tip Tip
Some players may have immunity to AFK kicks. Check the server page for rules.
:::
