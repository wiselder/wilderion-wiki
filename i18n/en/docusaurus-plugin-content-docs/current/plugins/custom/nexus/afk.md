# AFK System

:::tip Available on servers
**Overnight**
:::

The addon adds an AFK (Away From Keyboard) system that shows when a player is away from the computer.

---

## üí§ What is AFK?

**AFK** (Away From Keyboard) is a status showing that a player is temporarily absent. This helps other players understand why you're not responding in chat.

---

## üìã Commands

| Command | Description |
|---------|----------|
| `/afk` | Toggle AFK mode |

---

## ‚öôÔ∏è How it Works

### Manual Activation

Type `/afk` to manually enter AFK mode:
- A message about your status will appear in chat
- Your nickname may display with **AFK** tag

### Automatic Activation

On some servers AFK activates automatically if you don't move for a certain time.

### Exiting AFK

AFK mode automatically turns off when you:
- Move
- Write in chat
- Interact with the world
- Re-enter `/afk`

Upon exiting AFK you'll see a message with the time spent in absence mode.
