# AFK System

:::tip Available on servers
**Overnight**
:::

The addon adds an AFK (Away From Keyboard) system that shows when a player is away from the computer.

---

## üí§ What is AFK?

**AFK** (Away From Keyboard) is a status showing that a player is temporarily absent. This helps other players understand why you're not responding in chat.

---

## üìã Commands

| Command | Description |
|---------|----------|
| `/afk` | Toggle AFK mode |

---

## ‚öôÔ∏è How it Works

### Manual Activation

Type `/afk` to manually enter AFK mode:
- A message about your status will appear in chat
- Your nickname may display with **AFK** tag

### Automatic Activation

On some servers AFK activates automatically if you don't move for a certain time.

### Exiting AFK

AFK mode automatically turns off when you:
- Move
- Write in chat
- Interact with the world
- Re-enter `/afk`

Upon exiting AFK you'll see a message with the time spent in absence mode.

---

## ‚ö†Ô∏è AFK Kick

:::warning Warning
On some servers players may be kicked for being AFK too long!
:::

**Why this is needed:**
- Freeing slots for active players
- Reducing server load
- Fair resource distribution

**How to avoid kick:**
- Move periodically
- Don't leave the game unattended for long

:::tip Tip
Some players may have immunity to AFK-kick. Check the rules on the server page.
:::

## ‚ùì FAQ

<details>
<summary>**Do other players see my AFK status?**</summary>

Yes, your AFK status may display in the TAB-list or next to your nickname in chat.

</details>

<details>
<summary>**Can I write in chat while AFK?**</summary>

Technically yes, but sending a message will automatically take you out of AFK mode.

</details>

<details>
<summary>**How long until I get kicked for AFK?**</summary>

Time depends on server settings. Usually it's 10-30 minutes. Check on the specific server page.

</details>

<details>
<summary>**Does AFK work during fishing?**</summary>

AFK-fishing usually doesn't protect from kick, as it's considered inactive gameplay.

</details>
