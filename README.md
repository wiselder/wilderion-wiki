# Wilderion Wiki

Документация проекта Wilderion, построенная с помощью [Docusaurus](https://docusaurus.io/).

## Установка

```bash
npm install
```

## Локальная разработка

```bash
npm run start
```

Эта команда запускает локальный dev-сервер. Большинство изменений отображаются в реальном времени без перезапуска сервера.

## Сборка

```bash
npm run build
```

Эта команда генерирует статический контент в директорию `build`.

## Деплой

Сайт автоматически деплоится на GitHub Pages через GitHub Actions при пуше в ветку `main`.
